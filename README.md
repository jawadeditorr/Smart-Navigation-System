# Smart Navigation System

## Overview
A graph-based navigation system developed in C++ that finds the shortest route between cities using Dijkstra’s Algorithm.

## Technologies
- C++
- STL
- Graphs
- Priority Queues

## Features
- Shortest path calculation
- Dynamic city connections
- Efficient route finding
- Path reconstruction

## Algorithm Used
Dijkstra’s Algorithm

## Future Improvements
- GUI integration
- Real map APIs
- Traffic simulation
